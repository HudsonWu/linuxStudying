# 一些有用且有趣的命令

1. autojump <br/>
命令行跳转工具, 帮助快速从目录访问历史中的各个目录中迅速跳转 <br/>

2. thefuck <br/>
命令纠错 <br/>

3. tig <br/>
命令行下查看git历史提交记录的工具 <br/>

4. git summary <br/>
大致了解每个人对这个项目提交的commit数量和贡献度<br/>

5. ydict <br/>
命令行方式的有道词典<br/>
<https://github.com/TimothyYe/ydict> <br/>

6. er <br/>
命令行下查询和换算货币汇率 <br/>
<https://github.com/TimothyYe/exchangerate> <br/>
